# DevOps-scripts

1. Port Scanner
✅ Purpose:
Check for open TCP ports on a given IP or hostname.

📋 Description:
Helps detect which ports are open on a machine—useful for troubleshooting firewalls or verifying service availability.

🛠️ How to Use:
python port_scanner.py

🧪 Example Output:

Port 22 is open
Port 80 is open
